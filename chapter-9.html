Όψεις (Views)

Δημιουργία εικονικών πινάκων για ευκολότερη διαχείριση

9.1 CREATE VIEW

CREATE VIEW european_cities AS SELECT * FROM City WHERE CountryCode IN ( SELECT Code FROM Country WHERE Continent = 'Europe' );

9.2 Χρήση Views

SELECT * FROM european_cities;

9.3 DROP VIEW

DROP VIEW european_cities;
