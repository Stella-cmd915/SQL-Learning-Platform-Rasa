ΚΕΦΑΛΑΙΟ 03

Ερωτήματα Επιλογής

Χρήση της SELECT για ανάκτηση και φιλτράρισμα δεδομένων

3.1 Η Εντολή SELECT

Η SELECT είναι η βασικότερη εντολή της SQL για ανάκτηση δεδομένων.

-- Επιλογή όλων των πεδίων SELECT * FROM City; -- Επιλογή συγκεκριμένων πεδίων SELECT Name, Population FROM City;

3.2 WHERE - Φιλτράρισμα Δεδομένων

Η πρόταση WHERE χρησιμοποιείται για φιλτράρισμα εγγραφών:

SELECT * FROM City WHERE Population > 1000000;

Τελεστές Σύγκρισης:

  • = Ίσο
  • > Μεγαλύτερο
  • < Μικρότερο
  • >= Μεγαλύτερο ή ίσο
  • <= Μικρότερο ή ίσο
  • <> ή != Διάφορο

3.3 ORDER BY - Ταξινόμηση

-- Αύξουσα (ASC - default) SELECT Name FROM City ORDER BY Population ASC; -- Φθίνουσα (DESC) SELECT Name FROM City ORDER BY Population DESC;

3.4 LIKE - Pattern Matching

Για αναζήτηση με μοτίβα χρησιμοποιούμε LIKE:

-- Πόλεις που αρχίζουν με 'A' SELECT Name FROM City WHERE Name LIKE 'A%'; -- Πόλεις που τελειώνουν σε 'ς' SELECT Name FROM City WHERE Name LIKE '%ς'; -- Όνομα περιέχει 'θην' SELECT Name FROM City WHERE Name LIKE '%θην%';

3.5 DISTINCT - Μοναδικές Τιμές

-- Όλες οι ηπείρους (χωρίς επαναλήψεις) SELECT DISTINCT Continent FROM Country;

3.6 LIMIT - Περιορισμός Αποτελεσμάτων

-- Μόνο τις 10 πρώτες εγγραφές SELECT * FROM City LIMIT 10; -- Από τη 20η έως τη 30η εγγραφή SELECT * FROM City LIMIT 20, 10;
